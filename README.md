# Gesture-Controlled TurtleBot (ROS 2)

Control a TurtleBot using real-time hand gestures with OpenCV + Mediapipe in ROS 2.

## Features
- 🖐 Open palm → Stop
- ✊ Fist → Move forward
- ✌ Peace sign → Move backward
- 👉 Thumb → Turn right
- 👈 Index → Turn left

## Run

```bash
ros2 run gesture_controller gesture_control


Dependencies

    OpenCV

    Mediapipe

    ROS 2 (Humble)
